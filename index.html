🔔 Discord 通知設定

📢 統一通知設定 (推薦)

設定一個 Discord Webhook，所有 BOSS 擊殺都會發送到同一個頻道。適合個人使用或團隊協作。

🔗
🌐

✅ 啟用統一通知的優點：

  • 所有 BOSS 擊殺記錄自動發送到同一個頻道
  • 設定一次，終身受用
  • 適合團隊協作，所有成員都能即時看到
  • 不需要為每個 BOSS 單獨設定

🎯 個別 BOSS 通知設定 (進階)

為特定 BOSS 設定專屬的 Discord Webhook。這些設定會覆蓋統一通知（但兩個都會發送，如果都有設定）。

💡 個別設定的使用場景：

  • 重要 BOSS 發送到專屬頻道
  • 不同類型的 BOSS 分別發送到不同頻道
  • 某些 BOSS 需要 @特定角色 通知
  • 測試用途，避免打擾主頻道

📋 已設定的 Webhook 一覽

🔗 Webhook 設定備份與還原

備份統一 Webhook 與所有 BOSS 個別 Webhook 設定。更換瀏覽器或裝置時可快速還原，不需重新設定。

📤

匯出 Webhook 備份

將統一 Webhook 與所有 BOSS 個別 Webhook 儲存為 .json 檔。

📥

匯入 Webhook 備份

選擇先前下載的 Webhook 備份檔案以還原 Webhook 設定。

📋 包含：統一 Webhook URL、所有 BOSS 個別 Webhook URL

📖 如何取得 Discord Webhook？

  1. 在 Discord 伺服器中，選擇一個文字頻道
  2. 點選頻道設定 (齒輪圖示)
  3. 選擇「整合」→「Webhooks」
  4. 點選「新 Webhook」
  5. 複製 Webhook URL 並貼到上方欄位

✨ Discord 通知功能特色：

  • ✅ 即時發送 BOSS 擊殺記錄
  • ✅ 顯示擊殺時間、頻道、地圖資訊
  • ✅ 顯示預計重生時間範圍
  • ✅ 使用 BOSS 專屬顏色的訊息嵌入

⚠️ 注意事項：

  • 請勿將 Webhook 網址分享給他人
  • Webhook 設定僅儲存在你的瀏覽器中
  • 清除瀏覽器資料會同時清除 Webhook 設定
  • 個別設定會覆蓋統一通知（但兩個都會發送，如果都有設定）

💡 使用建議：

  • 個人玩家：只設定統一通知即可
  • 團隊合作：設定統一通知到團隊頻道
  • 精細管理：重要 BOSS 設定個別通知，其他用統一
  • 多重通知：兩種都設定，確保不會錯過任何通知
